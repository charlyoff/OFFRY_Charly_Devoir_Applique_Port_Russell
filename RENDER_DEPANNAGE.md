## Note Render gratuit

Juste apres la creation ou un redeploiement, Render gratuit peut mettre 1 a 2
minutes a stabiliser l'application. Si besoin :

1. attendre 1 a 2 minutes ;
2. faire Ctrl + F5 ;
3. sur Render, lancer `Manual Deploy -> Deploy latest commit`.

Le bon signe est d'arriver sur la page `Tableau de bord`.
